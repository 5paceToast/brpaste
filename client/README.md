Burning Rubber Paste
====================

Client Side Utilities
---------------------

### `brpaste.anypaste`
An [anypaste](https://anypaste.xyz/) plugin for `brpaste`.

### `brpaste.sh`
A shell script to paste to `brpaste`.

### `brpaste.zsh`
A zsh-style script/function-definition combo for pasting to `brpaste`.
